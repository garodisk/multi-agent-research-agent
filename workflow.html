Human-in-the-Loop Interrupt

When the Clarity Agent detects a vague query with no company name, langgraph.types.interrupt() pauses the graph and surfaces a clarification prompt to the caller. Execution resumes via Command(resume=...) — the Clarity Agent re-runs with the updated context, then routes to Research as normal. Conversation state is fully preserved across the interrupt boundary via MemorySaver.

Agents

Clarity Agent
src/agents/clarity.py
Determines whether the user's query names a specific company and has understandable intent. Uses LLM classification. Triggers interrupt() when the query is too vague. Resets research_attempts to 0 each turn.
State reads
query
State writes
clarity_status research_attempts = 0
Routes to
INTERRUPT → Research
Research Agent
src/agents/research.py
Calls the Tavily Search API for live web results on company news and financials, then merges with curated mock data. Synthesizes findings and assigns a confidence_score from 0 to 10 based on result quality.
State reads
query
State writes
research_findings confidence_score
Routes to
Validator (low) → Synthesis (high)
Validator Agent
src/agents/validator.py
Evaluates whether research findings are sufficient to fully answer the user's query. Increments the attempt counter and sends the graph back to Research up to 3 times before forcing Synthesis.
State reads
query research_findings research_attempts
State writes
validation_result research_attempts + 1
Routes to
Research (loop) → Synthesis
Synthesis Agent
src/agents/synthesis.py
Produces the final user-facing response. Passes the full messages conversation history to the LLM so follow-up questions are answered in context without re-asking for company names.

query research_findings messages (history)
State writes
messages += AIMessage

Conditional Routing

From To Condition
Clarity INTERRUPT No company name detected — query is too vague
Clarity Research clarity_status = "clear"
Research Validator confidence_score < 6 — findings may be incomplete
Research Synthesis confidence_score ≥ 6 — fast path, skips validation
Validator Research validation_result = "insufficient" AND attempts < 3
Validator Synthesis validation_result = "sufficient" OR attempts ≥ 3

State Schema

Field Type Description
query str The user's raw input text for the current turn
messages Annotated[list, add_messages] Full conversation history; LangGraph reducer appends each new message
clarity_status str "clear" or "needs_clarification" — set by Clarity Agent
research_findings str Formatted string of Tavily results + mock data from Research Agent
confidence_score int 0–10 quality score assigned by Research Agent; gates validator bypass
validation_result str "sufficient" or "insufficient" — set by Validator Agent
research_attempts int Loop counter reset to 0 by Clarity Agent; incremented by Validator Agent

Tools & Integrations

🕸️

LangGraph

StateGraph with conditional edges and MemorySaver checkpointer. Powers interrupt/resume and the validator retry loop.

🔎

Tavily

Live web search for company news, financials, and recent developments. Falls back to curated mock data when the API is unavailable.

🧩

OpenRouter

OpenAI-compatible gateway used for every LLM call across all agents. Model is configurable via MODEL_NAME env var.

💾

MemorySaver

In-memory LangGraph checkpointer keyed by thread_id. Accumulates the full message history across multi-turn conversations.

📈

LangSmith

Automatic end-to-end tracing when LANGCHAIN_API_KEY is set. Every agent node run and LLM call is captured in a trace.

▶️

Next.js UI

Chat frontend in frontend/. Streams responses from the FastAPI backend via SSE. Runs on localhost:3000.

How to Run

Run Locally (with your API keys)

# 1. clone and install
git clone https://github.com/garodisk/multi-agent-research-agent.git
cd multi-agent-research-agent
cd backend && uv sync && cd ..
npm install

# 2. create backend/.env
OPENROUTER_API_KEY=sk-or-...     # required
TAVILY_API_KEY=tvly-...          # optional
MODEL_NAME=openai/gpt-4o-mini

# 3. create .env.local at repo root
BACKEND_URL=http://localhost:8000

Get an OpenRouter key at openrouter.ai (also lets you use GPT-4o, Claude, Gemini, Llama, etc.). Tavily is optional; without it the agent falls back to mock data only.

Start Both Services

# terminal 1 — backend
cd backend
uv run uvicorn api:app --port 8000

# terminal 2 — frontend
npm run dev

# open
http://localhost:3000

# optional: 5-turn regression test
cd backend
uv run python test_multiturn.py

Or skip installation and try the deployed demo at the top of this page. The regression script drives the graph through a scripted Apple → Tesla conversation and asserts recency behavior.
